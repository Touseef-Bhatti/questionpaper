Admin Panel (QPaperGen)

- Login: `admin/login.php`
- Dashboard: `admin/dashboard.php`
- Manage Classes: `admin/manage_classes.php`
- Manage Books: `admin/manage_books.php`
- Manage Chapters: `admin/manage_chapters.php`
- Manage Questions: `admin/manage_questions.php`
- Manage Users: `admin/users.php`
- Settings: `admin/settings.php`

To grant access, add a row in `users` with role `admin` or `superadmin`.
